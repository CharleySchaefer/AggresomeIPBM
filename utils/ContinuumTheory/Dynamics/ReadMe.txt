Software Requirements:
---------------------
-Octave or Matlab (solve equations)
-Gnuplot          (plot figure)
-Terminal         (run .sh shell scripts)

Description:
-----------
Calculations are carried out using DiffusionDegradation.m (octave or matlab).
Running this script generates the Distributions_*.txt and SizeVsTime_*.txt data.
This data is plotted using the bash scripts mkfig_*.sh using gnuplot.
